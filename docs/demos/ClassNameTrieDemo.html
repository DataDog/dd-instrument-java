About the encoding
Each node is laid out as: [branchCount, key₀..keyₙ, val₀..valₙ, jump₁..jumpₙ]. Branch 0's segment immediately follows the node so its jump offset is omitted. A value's high bits encode markers — 0x8000 leaf, 0x4000 bud, 0x2000 glob — with the result number in the low 13 bits (0x1FFF). Jump offsets that don't fit in a char use the long-jump table via 0x8000.

Mappings

One entry per line: <number> <class-name>. Suffix * = glob.
